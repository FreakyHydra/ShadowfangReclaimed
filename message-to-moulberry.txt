Subject: Axiom running on Folia 26.1.2

Hi Moulberry,

I'm running a Folia 26.1.2 server (latest Minecraft) and wanted to get
Axiom working on it. Since Folia uses region threads instead of a single
server thread, I built a small compatibility bridge (RosettaStone) and
made a few targeted edits to Axiom's source:

- AxiomPaper.java — replaced Bukkit's scheduleSyncRepeatingTask with
  RosettaStone's FoliaCompat.scheduleSyncRepeating which routes to
  Folia's GlobalRegionScheduler

- Several packet listeners — wrapped region-bound operations with
  RegionDispatcher.runAtChunk() to ensure they execute on the correct
  region thread

The approach works well — Axiom runs stable on Folia 26.1.2 with these
changes. Just wanted to share in case you're interested. Happy to go
into more detail or send a diff if you'd like.

Server's at https://github.com/FreakyHydra/ShadowfangReclaimed if you
want to check it out.
